# 学习笔记


## 二叉树的前、中、后序遍历
1. 前序遍历：递归、使用栈存储父节点，与中序的区别为存储的同时便已经遍历该节点。
2. 中序遍历：递归、使用栈存储父节点。
3. 后序遍历：递归、使用双栈法、使用一个栈，父节点以null来代替。

## 堆和二叉堆
1. 二叉堆一般都通过“数组”来实现。
2. 二叉堆是堆的一种常见且简单的实现，不是堆的最优实现。
3. 在实际使用时，堆在C++中可以使用priority_queue解决问题。
